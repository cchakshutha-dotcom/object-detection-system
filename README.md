# Object Detection System

## Overview
This project is a real-time Object Detection System developed using YOLOv8, OpenCV, and Python.

The system detects multiple objects from webcam video in real time and displays detected object names along with confidence scores.

This project demonstrates the use of Artificial Intelligence, Computer Vision, and Deep Learning techniques for smart real-time object recognition applications.

---

## Features
- Real-time object detection
- Webcam-based live detection
- YOLOv8 object recognition
- Confidence score display
- Fast frame processing
- Real-time visualization using OpenCV
- Multiple object detection support

---

## Technologies Used
- Python
- YOLOv8
- OpenCV
- NumPy
- Computer Vision
- Deep Learning

---

## Project Structure
- `object_detection.py` → Main program for object detection
- `best.pt` → Trained YOLO model weights
- `yolov8n.pt` → YOLOv8 pre-trained model weights

---

## How the Project Works
1. Webcam captures live video frames.
2. YOLOv8 processes each frame.
3. Objects are detected in real time.
4. Bounding boxes are drawn around detected objects.
5. Object names and confidence scores are displayed on screen.
